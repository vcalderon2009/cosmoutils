Documentation
=============

`Cosmo-Utils <https://github.com/vcalderon2009/cosmoutils>`_ is a package 
used in many of my projects related to LSS (large-scale structure), 
astrophysics, and the `Sloan Digital Sky Survey <http://www.sdss.org/>`_.

The source code is freely availabe at
`https://github.com/vcalderon2009/cosmoutils <https://github.com/vcalderon2009/cosmoutils>`_.
